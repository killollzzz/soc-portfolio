### Шаг 5: Beaconing

Обнаружен периодический трафик к C2-серверам. Интервал между GET-запросами к одному IP составляет **~88-94 секунды**. Это классический beaconing — вредонос регулярно опрашивает сервер на наличие команд.

**Пример (76.223.54.146, /5l58/):**

text

GET → +88s → GET → +94s → GET → +88s → GET → +93s → GET → +88s → GET → +90s → GET → +88s → GET

### Шаг 6: Передача данных

POST-запросы с `Content-Type: application/x-www-form-urlencoded`. Размеры: ~170-240 байт (малые POST, регистрация), ~700-770 байт (средние POST), ~1300-1400 байт (большие POST, передача данных жертвы).
